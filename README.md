## Advanced data structures implementation in C#

	A personal C# library of advanced data structures,
built from scratch as each one was learned. 
No libraries, no shortcuts — every structure is implemented manually to build a deep understanding 
of how they work internally.

What’s Implemented
∙	Min Heap and Max Heap
∙	Min Priority Queue and Max Priority Queue
∙	Red-Black Tree
∙	Graph — including shortest path using Dijkstra’s Algorithm
∙	Tree Problem Solving
∙	Basic Searching Algorithms
∙	Simple Sorting Algorithms
